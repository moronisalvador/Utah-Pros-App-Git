UPR Tech Design Standard — Direction B "Apple Field Pro"

The rendered system of record for the /tech/* greenfield redesign. Locked and proven on four screens (dashboard, job hub, drying module, plus the flow challenger's ripened Finish). Everything on this page is token-driven and flips with the theme control above — both themes are designed with equal care.

0Principles

Ink is the brand, color is state

Black/ink identity everywhere. Chromatic color appears ONLY as the 5 field statuses (amber OMW, green working, red paused, blue scheduled, gray done), the semantic good/warn/critical/info/neutral family they double as, and the focus ring. Everything else is ink. Banned: side-stripe status borders, gradient accents, colored headings, tinted cards as status.

Persona law: the 64-year-old gloved tech

Standing in a flooded basement, gloves on, phone in one hand, direct sunlight. Primary field actions are 48px; 44px only as documented-secondary with a comment at the declaration site; hit areas under 24px are banned outright. Type floors: 11px absolute (badge numerals and column headers only, never actionable), 12px for anything tappable. Primary text leans 7:1 or better. Every status carries a redundant shape glyph so state survives grayscale.

Glance and gate

The dashboard surfaces cross-job attention; the hub glances one job; the module goes deep. A module's full instrument never renders inside the hub — one gateway row with mirrored aggregate numbers gates in, at most one escalation row sits beneath it, and the gate destination owns all actions.

One primary action per screen

The black pill owns it. The FAB, docks, quiet buttons, and rows are subordinate. Never two black pills visible at once. Destructive or terminal actions arm with a two-tap confirm in place — never a modal, never a browser confirm.

Soft, layered, committed

Neutral layered surfaces (never blue-tinted or cream), exactly two shadow levels, large continuous radii with inner-always-smaller nesting, ONE frosted surface — the tab bar. Hairlines divide within a surface; shadows float a surface. Never both on the same edge.

Quiet instruments

Numbers a tech compares are 17px+ tabular ink at weight 650; locators and timestamps are 13px muted. Summaries are one quiet numbers card with exactly one emphasized cell. Motion is frequency-tiered: high-frequency controls are instant — deleting the animation is the correct craft call.

1Color

Every token, both themes, side by side. Dark is graphite-elevated and strictly neutral — never blue-tinted; cream and beige are banned. Dark status backgrounds are alpha tints of the status fg (shown here composited over the dark card surface #1E1F21). Contrast ratios are computed WCAG 2.x values from the foundations spec — nothing in either theme falls below 4.5:1.

Theme mechanics

Light values + color-scheme: light live on .tech-layout; [data-theme="dark"] .tech-layout re-declares the SAME token names with dark values + color-scheme: dark (native form controls, scrollbars, and keyboard come free). ThemeContext (KEEP) stamps data-theme on the html element; mode persists in upr_theme_pref; system follows prefers-color-scheme live; system-auto is the proposed default. Components never know the theme — they pick token NAMES, the theme block swaps VALUES. Only /tech/* swaps; desktop is untouched. Capacitor status-bar theme-color points at the ground per theme (#F2F3F5 / #141416).

4Icons — duotone rounded, 24px grid

Drawn in INK (currentColor) — an icon never carries its own hex; chromatic color reaches a glyph only through a status container. Grid 24, live area about x/y 3.8 to 20.5, optical centering beats mathematical. Stroke 1.8 with round caps and joins; pure-line micro-glyphs (check, chevrons, trend, x, pulse) run stroke 2 so they hold at 13px; the FAB plus is the one 2.2 outlier. Duotone = the same path twice: a silhouette fill at opacity: var(--duo, .16) under a stroked outline; tiny solid accents (help dot, gauge hub) are full-opacity punctuation under about 2px radius. Corners are friendly: rects carry rx 2.8 to 3.6, organic forms use soft continuous curves — nothing in the family has a square corner. No emoji, ever — every pictograph in the product is an icon from this system.
